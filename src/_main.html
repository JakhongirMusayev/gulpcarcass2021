Sass quyidagi imkoniyatlarni beradi:
  1. CSS qoidalarini bir-birining ichiga joylashtirish;
    2. Bu ota selektorlarning takrorlanishi oldini oladi va katta maketlarda CSSni qo'llashni oconlashtiradi.

    SCSS

     
  #main p {
    color: #00ff00;
    width: 97%;
    .redbox {
      background-color: #ff0000; 
      color: #000000;
    }
  }

    CSS

     
    #main p {
      color: #00ff00;
      width: 97%;
    }
    #main p .redbox {
      background-color: #ff0000; 
      color: #000000;
    }
  2. Ota selektorga & belgisi orqali havola yaratadi.
    3. Bu uning qayta-qayta takrorlanmasligini ta'minlaydi.

    SCSS

     
  a {
    font-weight: bold;;
    text-decoration: none;;
    &:hover {
      text-decoration: underline; 
    }
    body.firefox & {
    font-weight: normal;;
    }
  }

    CSS

     
  a {
    font-weight: bold;;
    text-decoration: none;; 
  }
  a:hover {
    text-decoration: underline; 
  }
  body.firefox a {
  font-weight: normal;;
  }
  3. Bir xil iboralar bilan boshlanuvchi xossalarni umumlashtirish.
    4. Masalan, "font" bilan boshlanuvchi xossalar.

    SCSS

     
  .funky {
    font: {
    family: fantasy;; 
    weight: bold;;
    size: 30em;;
    }
  }

    CSS

     
  .funky {
    font-family: fantasy;; 
    font-weight: bold;;
    font-size: 30em;;
  }

    Само пространство имен также может иметь значение. Например:

    SCSS

     
  .funky {
    font: 20px/24px fantasy { 
    weight: bold;;
    }
  }

    CSS

     
  .funky {
    font: 20px/24px fantasy;; 
    font-weight: bold;;
  }

    SCSS

     
  .funky {
    margin: auto{ 
    top: 10px;;
    bottom: 15px;; 
    }
  }

    CSS

     
  .funky {
    margin: auto;;
    margin-top: 10px;;
    margin-bottom: 15px;; 
  }
  4. Shablon selektorlaridan foydalanish.
    5. Selectorning xossa va qiymatlarining shablonini yaratish. Lekin yodda tutish kerakki, u "@extend"siz ishlamaydi.

    SCSS

     
  %pr-type {
    border: 1px solid brown;  
    *border: 3px dotted yellow; 
    color: chocolate;  
  }

  %pr-for-type {
    height: 15px; 
    padding-left: 10px; 
    margin-bottom: 20px; 
  }

  section.shablon-selectors div p { 
    @extend %pr-type 
  }

    CSS

     
  section.shablon-selectors div p { 
    border: 1px solid brown; { 
    *border: 3px dotted yellow; 
    color: chocolate; 
  }
  5. Sharhlarning (Comments) o'ziga xosligi.
    6. Masalan: CSS faqat ko'psatrli sharhlarni ishlata oladi, SCSS esa bir satrlilarni ham..

    SCSS

     
  
    /* Это многострочный комментарий.
     * Так как этот тип комментирования
     * поддерживается в CSS, то он
     * попадет в скомпилированный CSS-файл. */ 
     body { 
      color: black;
    }
  
  
    // Это однострочный комментарий.
    // CSS не поддерживает данный вариант комментирования. 
    // Поэтому данного комментария не будет в CSS-файле.
    a { 
      color: green;
    }

    CSS

     
  
    /* Это многострочный комментарий.
     * Так как этот тип комментирования
     * поддерживается в CSS, то он
     * попадет в скомпилированный CSS-файл. */ 
  
    body { 
      color: black;
    }

    a { 
      color: green;
    }

    Agar ko'psatrli sharhning 1-simvoli "!" belgisi bo'lsa, u nafaqat CSS faylga o'tadi, balki CSS fayl siqilganda ham unda qoladi. Bu, masalan, mualliflik huquqlarini ko'rsatib ketishda asqotadi.

    SCSS

     
  
    body { 
      color: black;
    }
    /*! Bu ko'psatrli sharh bo'lib,
     * uning 1-simvoli "!"dir,
     * Shuning uchun u nafaqat CSS faylga
     * o'tadi, balki CSS fayl
     * siqilganda ham unda qoladi. */

    CSS

     
  
    body{color: black;}/*! Bu ko'psatrli sharh bo'lib,
    * uning 1-simvoli "!"dir,
    * Shuning uchun u nafaqat CSS faylga
    * o'tadi, balki CSS fayl 
    * siqilganda ham unda qoladi. */
  
  (Yulduzchaliklari shu kabi alohida qatorlardan joy oladi)

    Поскольку многострочные комментарии становятся частью результирующего CSS, в них разрешена интерполяция. Например:

    SCSS

     
  
    $version: "1.1.3";
    /* Этот CSS построен с использованием SomeGrids.
    Версия: #{$version}. */

    CSS

     
  
    /* Этот CSS построен с использованием SomeGrids. 
    Версия: 1.1.3. */
